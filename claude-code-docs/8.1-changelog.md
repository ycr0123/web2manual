# 변경 이력

## 2.1.45

- Claude Sonnet 4.6 지원 추가
- `--add-dir` 디렉토리에서 `enabledPlugins` 및 `extraKnownMarketplaces` 읽기 지원 추가
- `spinnerTipsOverride` 설정 추가로 스피너 팁 사용자 지정 가능 — 사용자 지정 팁 문자열 배열로 `tips`를 구성하고, 선택적으로 `excludeDefault: true`를 설정하여 기본 팁 대신 사용자 지정 팁만 표시
- SDK에 `SDKRateLimitInfo` 및 `SDKRateLimitEvent` 타입 추가로 소비자가 활용률, 재설정 시간, 초과 정보를 포함한 속도 제한 상태 업데이트를 수신할 수 있게 됨
- tmux로 생성된 프로세스에 API 공급자 환경 변수를 전파하여 Bedrock, Vertex, Foundry에서 에이전트 팀 팀원이 실패하는 문제 수정 (#23561)
- macOS에서 올바른 사용자별 임시 디렉토리를 사용하여 임시 파일 작성 시 샌드박스 "operation not permitted" 오류 수정 (#21654)
- Task 도구 (백그라운드 에이전트)가 완료 시 `ReferenceError`로 충돌하는 문제 수정 (#22087)
- 이미지가 입력 필드에 붙여넣기된 경우 Enter를 눌렀을 때 자동 완성 제안이 수락되지 않는 문제 수정
- 하위 에이전트가 호출한 스킬이 압축 후 메인 세션 컨텍스트에 잘못 표시되는 문제 수정
- 시작 시마다 과도한 `.claude.json.backup` 파일이 누적되는 문제 수정
- 재시작 없이 설치 후 플러그인이 제공하는 명령, 에이전트, 훅이 즉시 사용 가능하지 않은 문제 수정
- 통계 캐싱을 위한 세션 기록의 즉시 로딩 제거로 시작 성능 향상
- 대용량 출력을 생성하는 셸 명령의 메모리 사용량 개선 — 명령 출력 크기에 비례하여 RSS가 무한정 증가하지 않음
- 활성 처리 중인 파일이나 검색 패턴을 요약 줄 아래에 표시하도록 접힌 읽기/검색 그룹 개선
- [VSCode] 프로젝트/사용자/세션에 대한 권한 대상 선택이 세션 간 유지되도록 개선

- 로그인 중 브라우저가 자동으로 열리지 않을 때 OAuth URL을 복사하기 위한 'c' 키보드 단축키 추가
- JavaScript 템플릿 리터럴 (`${index + 1}` 등)을 포함하는 heredoc이 있는 bash 명령을 실행할 때 충돌 수정
- REPL이 완전히 준비되기 전에 입력된 키 입력을 캡처하도록 시작 개선
- 수락 시 텍스트를 삽입하는 대신 제거 가능한 첨부 파일로 표시하도록 파일 제안 개선
- [VSCode] 설치 수 추가
