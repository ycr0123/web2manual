# 인증 설정

> 조직에서 Claude Code의 사용자 인증 및 자격 증명 관리를 구성하는 방법을 알아봅니다.

## 인증 방법

Claude Code 설정에는 Anthropic 모델에 대한 접근이 필요합니다. 팀의 경우 다음 방법 중 하나로 Claude Code 접근을 설정할 수 있습니다:

* [Claude for Teams 또는 Enterprise](#claude-for-teams-or-enterprise) (권장)
* [Claude Console](#claude-console-authentication)
* [Amazon Bedrock](/en/amazon-bedrock)
* [Google Vertex AI](/en/google-vertex-ai)
* [Microsoft Foundry](/en/microsoft-foundry)

### Claude for Teams 또는 Enterprise

[Claude for Teams](https://claude.com/pricing#team-&-enterprise)와 [Claude for Enterprise](https://anthropic.com/contact-sales)는 Claude Code를 사용하는 조직에게 최상의 경험을 제공합니다. 팀원들은 중앙화된 청구와 팀 관리로 Claude Code와 웹의 Claude 모두에 접근할 수 있습니다.

* **Claude for Teams**: 협업 기능, 관리자 도구, 청구 관리가 있는 셀프 서비스 플랜. 소규모 팀에 적합합니다.
* **Claude for Enterprise**: SSO, 도메인 캡처, 역할 기반 권한, 규정 준수 API, 조직 전체 Claude Code 구성을 위한 관리형 정책 설정을 추가합니다. 보안 및 규정 준수 요구 사항이 있는 대규모 조직에 적합합니다.

**단계:**

1. [Claude for Teams](https://claude.com/pricing#team-&-enterprise)를 구독하거나 [Claude for Enterprise](https://anthropic.com/contact-sales)를 위해 영업팀에 문의하세요.
2. 관리자 대시보드에서 팀원을 초대하세요.
3. 팀원들이 Claude Code를 설치하고 Claude.ai 계정으로 로그인합니다.

### Claude Console 인증

API 기반 청구를 선호하는 조직의 경우 Claude Console을 통해 접근을 설정할 수 있습니다.

**단계:**

1. 기존 Claude Console 계정을 사용하거나 새로 만드세요.
2. 다음 방법 중 하나로 사용자를 추가하세요:
   * Console에서 사용자를 대량 초대 (Console → 설정 → 멤버 → 초대)
   * [SSO 설정](https://support.claude.com/en/articles/13132885-setting-up-single-sign-on-sso)
3. 사용자 초대 시 다음 중 하나를 할당하세요:
   * **Claude Code** 역할: 사용자가 Claude Code API 키만 만들 수 있음
   * **개발자** 역할: 사용자가 모든 종류의 API 키를 만들 수 있음
4. 초대된 각 사용자가 수행해야 할 작업:
   * Console 초대 수락
   * [시스템 요구 사항 확인](/en/setup#system-requirements)
   * [Claude Code 설치](/en/setup#installation)
   * Console 계정 자격 증명으로 로그인

### 클라우드 제공업체 인증

Amazon Bedrock, Google Vertex AI, 또는 Microsoft Azure를 사용하는 팀의 경우:

1. [Bedrock 문서](/en/amazon-bedrock), [Vertex 문서](/en/google-vertex-ai), 또는 [Microsoft Foundry 문서](/en/microsoft-foundry)를 따르세요.
2. 클라우드 자격 증명 생성을 위한 환경 변수와 지침을 사용자에게 배포하세요. [구성 관리 방법](/en/settings)에 대해 자세히 알아보세요.
3. 사용자들이 [Claude Code를 설치](/en/setup#installation)할 수 있습니다.

## 자격 증명 관리

Claude Code는 인증 자격 증명을 안전하게 관리합니다:

* **저장 위치**: macOS에서 API 키, OAuth 토큰 및 기타 자격 증명은 암호화된 macOS 키체인에 저장됩니다.
* **지원되는 인증 유형**: Claude.ai 자격 증명, Claude API 자격 증명, Azure Auth, Bedrock Auth, Vertex Auth.
* **커스텀 자격 증명 스크립트**: [`apiKeyHelper`](/en/settings#available-settings) 설정을 사용하여 API 키를 반환하는 셸 스크립트를 실행하도록 구성할 수 있습니다.
* **새로 고침 간격**: 기본적으로 `apiKeyHelper`는 5분 후 또는 HTTP 401 응답 시 호출됩니다. 커스텀 새로 고침 간격을 위해 `CLAUDE_CODE_API_KEY_HELPER_TTL_MS` 환경 변수를 설정하세요.
