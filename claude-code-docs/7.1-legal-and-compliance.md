# 법적 사항 및 규정 준수

> Claude Code에 관한 법적 계약, 규정 준수 인증, 보안 정보입니다.

## 법적 계약

### 라이선스

Claude Code 사용은 다음을 따릅니다.

* [상업 약관](https://www.anthropic.com/legal/commercial-terms) - Teams, Enterprise, Claude API 사용자
* [소비자 서비스 약관](https://www.anthropic.com/legal/consumer-terms) - Free, Pro, Max 사용자

### 상업 계약

Claude API를 직접 사용(1P)하거나 AWS Bedrock 또는 Google Vertex를 통해 액세스(3P)하는 경우, 상호 합의가 없는 한 기존 상업 계약이 Claude Code 사용에 적용됩니다.

## 규정 준수

### 의료 규정 준수 (BAA)

고객이 당사와 Business Associate Agreement (BAA)를 체결하고 Claude Code를 사용하려는 경우, 고객이 BAA를 체결하고 Zero Data Retention (ZDR)을 활성화한 경우 BAA가 자동으로 Claude Code를 커버하도록 확장됩니다. BAA는 Claude Code를 통해 흐르는 해당 고객의 API 트래픽에 적용됩니다.

## 사용 정책

### 허용 가능한 사용

Claude Code 사용은 [Anthropic 사용 정책](https://www.anthropic.com/legal/aup)을 따릅니다. Pro 및 Max 플랜에 대해 광고된 사용 한도는 Claude Code 및 Agent SDK의 일반적인 개인 사용을 전제로 합니다.

### 인증 및 자격 증명 사용

Claude Code는 OAuth 토큰 또는 API 키를 사용하여 Anthropic 서버에 인증합니다. 이러한 인증 방법은 서로 다른 목적으로 사용됩니다.

* **OAuth 인증** (Free, Pro, Max 플랜에서 사용)은 Claude Code 및 Claude.ai에만 사용됩니다. Claude Free, Pro 또는 Max 계정을 통해 얻은 OAuth 토큰을 [Agent SDK](https://platform.claude.com/docs/en/agent-sdk/overview)를 포함한 다른 제품, 도구 또는 서비스에 사용하는 것은 허용되지 않으며 [소비자 서비스 약관](https://www.anthropic.com/legal/consumer-terms)을 위반합니다.
* **개발자**는 [Agent SDK](https://platform.claude.com/docs/en/agent-sdk/overview)를 사용하는 것을 포함하여 Claude 기능과 상호 작용하는 제품이나 서비스를 구축하는 경우 [Claude Console](https://platform.claude.com/) 또는 지원되는 클라우드 공급자를 통해 API 키 인증을 사용해야 합니다. Anthropic은 서드파티 개발자가 Claude.ai 로그인을 제공하거나 사용자를 대신하여 Free, Pro 또는 Max 플랜 자격 증명을 통해 요청을 라우팅하는 것을 허용하지 않습니다.

Anthropic은 이러한 제한 사항을 사전 통보 없이 시행할 수 있는 조치를 취할 권리를 보유합니다.

사용 사례에 대한 허용된 인증 방법에 대한 질문은 [영업팀에 문의](https://www.anthropic.com/contact-sales)하세요.
