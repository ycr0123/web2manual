# 터미널 설정 최적화

> 터미널이 올바르게 설정되어 있을 때 Claude Code가 최상으로 작동합니다. 다음 가이드라인에 따라 경험을 최적화하세요.

### 테마 및 외관

Claude는 터미널의 테마를 제어할 수 없습니다. 이는 터미널 애플리케이션에서 처리됩니다. `/config` 명령어를 통해 언제든지 Claude Code의 테마를 터미널에 맞출 수 있습니다.

Claude Code 인터페이스 자체의 추가적인 사용자 정의를 위해, 터미널 하단에 현재 모델, 작업 디렉토리, git 브랜치와 같은 컨텍스트 정보를 표시하는 커스텀 상태 표시줄을 구성할 수 있습니다.

### 줄 바꿈

Claude Code에서 줄 바꿈을 입력하는 몇 가지 옵션이 있습니다:

* **빠른 이스케이프**: `\` 다음에 Enter를 입력하여 새 줄 생성
* **Shift+Enter**: iTerm2, WezTerm, Ghostty, Kitty에서 기본으로 작동
* **키보드 단축키**: 다른 터미널에서 새 줄을 삽입하는 키 바인딩 설정

**다른 터미널을 위한 Shift+Enter 설정**

Claude Code 내에서 `/terminal-setup`을 실행하여 VS Code, Alacritty, Zed, Warp용 Shift+Enter를 자동으로 구성하세요.

> `/terminal-setup` 명령어는 수동 구성이 필요한 터미널에서만 표시됩니다. iTerm2, WezTerm, Ghostty, Kitty를 사용 중인 경우 Shift+Enter가 이미 기본으로 작동하므로 이 명령어가 표시되지 않습니다.

**Option+Enter 설정 (VS Code, iTerm2 또는 macOS Terminal.app)**

**Mac Terminal.app의 경우:**

1. 설정 → 프로파일 → 키보드 열기
2. "Option을 Meta 키로 사용" 체크

**iTerm2 및 VS Code 터미널의 경우:**

1. 설정 → 프로파일 → 키 열기
2. 일반 아래에서 왼쪽/오른쪽 Option 키를 "Esc+"로 설정

### 알림 설정

적절한 알림 구성으로 Claude가 작업을 완료할 때 절대 놓치지 마세요:

#### iTerm 2 시스템 알림

작업 완료 시 iTerm 2 알림을 받으려면:

1. iTerm 2 기본 설정 열기
2. 프로파일 → 터미널로 이동
3. "벨 무음" 활성화 및 필터 알림 → "이스케이프 시퀀스 생성 알림 전송"
4. 선호하는 알림 지연 설정

이 알림은 iTerm 2에 특정하며 기본 macOS 터미널에서는 사용할 수 없습니다.

#### 커스텀 알림 훅

고급 알림 처리를 위해 자체 로직을 실행하는 알림 훅을 만들 수 있습니다.

### 큰 입력 처리

방대한 코드나 긴 지시로 작업할 때:

* **직접 붙여넣기 피하기**: Claude Code는 매우 긴 붙여넣기 내용으로 어려움을 겪을 수 있습니다
* **파일 기반 워크플로우 사용**: 내용을 파일에 작성하고 Claude에게 읽도록 요청하세요
* **VS Code 제한 사항 인식**: VS Code 터미널은 긴 붙여넣기를 잘라내는 경향이 있습니다

### Vim 모드

Claude Code는 `/vim`으로 활성화하거나 `/config`를 통해 구성할 수 있는 Vim 키 바인딩의 일부를 지원합니다.

지원되는 서브셋은 다음을 포함합니다:

* 모드 전환: `Esc` (NORMAL로), `i`/`I`, `a`/`A`, `o`/`O` (INSERT로)
* 탐색: `h`/`j`/`k`/`l`, `w`/`e`/`b`, `0`/`$`/`^`, `gg`/`G`, `f`/`F`/`t`/`T` (`;`/`,` 반복 포함)
* 편집: `x`, `dw`/`de`/`db`/`dd`/`D`, `cw`/`ce`/`cb`/`cc`/`C`, `.` (반복)
* 복사/붙여넣기: `yy`/`Y`, `yw`/`ye`/`yb`, `p`/`P`
* 텍스트 객체: `iw`/`aw`, `iW`/`aW`, `i"`/`a"`, `i'`/`a'`, `i(`/`a(`, `i[`/`a[`, `i{`/`a{`
* 들여쓰기: `>>`/`<<`
* 줄 작업: `J` (줄 합치기)

전체 참조는 인터랙티브 모드를 참조하세요.
