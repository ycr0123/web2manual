## Claude Code 데스크톱 앱

터미널 사용이 익숙하지 않거나, 더 편리한 GUI 환경을 선호한다면 Claude Code 데스크톱 앱을 사용할 수 있습니다.

### 데스크톱 앱 설치

**macOS:**

1. [claude.ai/download](https://claude.ai/download)에서 macOS용 앱을 다운로드합니다
2. 다운로드한 `.dmg` 파일을 열고 Applications 폴더로 드래그합니다
3. 앱을 처음 실행하면 보안 경고가 나타날 수 있습니다 - 시스템 설정에서 허용합니다

**Windows:**

1. 동일한 다운로드 페이지에서 Windows 설치 파일(`.exe`)을 다운로드합니다
2. 설치 프로그램을 실행하고 안내에 따라 설치합니다

<Callout type="note">
데스크톱 앱은 내부적으로 동일한 Claude Code CLI를 사용합니다. 터미널 버전과 기능적으로 동일하며, 단지 접근 방법이 다릅니다.
</Callout>

### 데스크톱 앱 주요 기능

| 기능 | 설명 |
|------|------|
| 빠른 실행 | 시스템 트레이 아이콘에서 바로 접근 |
| 멀티 세션 | 여러 프로젝트를 동시에 작업 |
| 히스토리 | 이전 대화 내용 조회 |
| 드래그 앤 드롭 | 파일을 직접 드래그하여 업로드 |
| 자동 업데이트 | 앱이 자동으로 최신 버전 유지 |

## VS Code 통합

VS Code에서 Claude Code를 사용하면 에디터와 AI 어시스턴트가 긴밀하게 통합됩니다.

### 확장 프로그램 설치

1. VS Code에서 **Extensions** 패널을 엽니다 (`Ctrl+Shift+X` 또는 `Cmd+Shift+X`)
2. 검색창에 **"Claude Code"**를 입력합니다
3. Anthropic이 배포한 공식 확장 프로그램을 설치합니다

또는 VS Code 명령 팔레트에서:

<CLIExample
  command="code --install-extension anthropic.claude-code"
  description="터미널에서 직접 VS Code 확장 프로그램을 설치합니다."
/>

### VS Code에서 Claude Code 사용하기

확장 프로그램 설치 후 VS Code에서 Claude Code를 시작하는 방법:

**방법 1: 사이드바 패널**
- 왼쪽 사이드바에서 Claude Code 아이콘을 클릭합니다
- 채팅 인터페이스가 나타납니다

**방법 2: 터미널 통합**
- VS Code의 내장 터미널을 열고 (`Ctrl+` `` ` ``)
- `claude` 명령어를 입력합니다

**방법 3: 키보드 단축키**
- `Ctrl+Shift+C` (macOS: `Cmd+Shift+C`)로 빠르게 Claude Code를 실행합니다

### VS Code 통합의 장점

<Callout type="tip">
VS Code에서 Claude Code를 사용할 때, 현재 열려있는 파일이나 선택된 코드를 자동으로 컨텍스트로 제공할 수 있습니다.
</Callout>

```
// 예시: 코드를 선택하고 Claude Code에게 설명 요청
function calculateTax(income, rate) {
  return income * rate;
}
```

코드를 선택한 상태에서 Claude Code를 열면:
- 선택된 코드가 자동으로 컨텍스트에 포함됩니다
- "이 함수를 설명해줘" 또는 "이 코드를 개선해줘" 등의 자연어 명령을 사용할 수 있습니다

## 개발 환경 최적화 팁

### 작업 디렉토리 설정

Claude Code는 실행된 디렉토리를 기준으로 작업합니다. 프로젝트 루트에서 실행하는 것이 가장 효과적입니다:

<CLIExample
  command="cd /path/to/your/project && claude"
  description="프로젝트 루트 디렉토리에서 Claude Code를 시작합니다."
/>

### CLAUDE.md 파일 생성

새 프로젝트에서 Claude Code를 처음 사용할 때 CLAUDE.md 파일을 생성하면 더 효율적으로 작업할 수 있습니다:

```markdown
# 프로젝트명

## 기술 스택
- Frontend: React 18, TypeScript
- Backend: Node.js, Express
- Database: PostgreSQL

## 코딩 규칙
- 함수형 컴포넌트 사용
- TypeScript strict 모드 활성화
- ESLint + Prettier 설정 준수

## 자주 사용하는 명령어
- `npm start`: 개발 서버 시작
- `npm test`: 테스트 실행
- `npm run build`: 프로덕션 빌드
```

<Collapsible title="JetBrains IDE에서의 설치 방법">
JetBrains의 IntelliJ IDEA, PyCharm, WebStorm 등에서도 Claude Code를 사용할 수 있습니다.

**플러그인 설치:**
1. File → Settings → Plugins로 이동합니다
2. Marketplace 탭에서 "Claude Code"를 검색합니다
3. Install 버튼을 클릭합니다
4. IDE를 재시작합니다

JetBrains 플러그인에 대한 자세한 내용은 "IDE 통합" 레슨에서 다룹니다.
</Collapsible>

## 설정 동기화

여러 컴퓨터에서 Claude Code를 사용한다면 설정을 동기화할 수 있습니다:

- **설정 파일 위치**: `~/.claude/` 디렉토리
- **동기화 방법**: Dropbox, iCloud, OneDrive 등 클라우드 스토리지 활용
- **팀 설정 공유**: 프로젝트의 CLAUDE.md를 Git에 커밋하여 공유
