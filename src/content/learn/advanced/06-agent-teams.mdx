## Agent Teams 구축

이 레슨은 준비 중입니다.

## Agent Teams 활성화

```bash
export CLAUDE_CODE_EXPERIMENTAL_AGENT_TEAMS=1
claude
```

## 팀 패턴

Agent Teams를 사용하면 여러 에이전트가 병렬로 작업을 수행할 수 있습니다.

<Callout type="warning">
Agent Teams는 실험적 기능입니다. 프로덕션 환경에서 사용하기 전에 충분히 테스트하세요.
</Callout>
