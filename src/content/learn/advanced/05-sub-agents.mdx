## Sub-agents 활용

이 레슨은 준비 중입니다.

## 서브에이전트 생성

서브에이전트는 `.claude/agents/` 디렉토리에 마크다운 파일로 정의합니다.

```markdown
---
name: code-reviewer
description: 코드 리뷰를 전문으로 하는 에이전트
tools: Read, Grep, Glob
---

# 코드 리뷰 에이전트

당신은 코드 리뷰 전문가입니다...
```

<Callout type="note">
서브에이전트는 Task() 도구를 통해 호출됩니다. 각 서브에이전트는 독립된 200K 토큰 컨텍스트를 가집니다.
</Callout>
