## 고급 기능 개요

이 레슨은 준비 중입니다.

## 고급 기능 생태계

Claude Code의 고급 기능들:

| 기능 | 용도 |
|------|------|
| Skills | 재사용 가능한 AI 지시사항 모듈 |
| Hooks | 이벤트 기반 자동화 |
| MCP | 외부 도구 통합 |
| Sub-agents | 복잡한 작업 분산 |
| Agent Teams | 다중 에이전트 협력 |
| Headless | CI/CD 자동화 |
| Plugins | 기능 묶음 배포 |
| Sandboxing | 보안 격리 환경 |

<Callout type="note">
고급 기능들은 서로 조합하여 사용할 때 가장 강력합니다. 예를 들어 Skills와 Hooks를 함께 사용하면 자동화된 코드 품질 관리 시스템을 구축할 수 있습니다.
</Callout>
