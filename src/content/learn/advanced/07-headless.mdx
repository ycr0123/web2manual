## Headless 모드 자동화

이 레슨은 준비 중입니다.

## GitHub Actions 예시

```yaml
name: AI Code Review
on: [pull_request]
jobs:
  review:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - run: npm install -g @anthropic-ai/claude-code
      - run: claude -p "변경된 파일들을 코드 리뷰해줘"
        env:
          ANTHROPIC_API_KEY: ${{ secrets.ANTHROPIC_API_KEY }}
```

<Callout type="warning">
CI/CD 환경에서는 반드시 --allowedTools 플래그를 사용하여 Claude Code가 실행할 수 있는 도구를 제한하세요. 보안을 위해 최소 권한 원칙을 적용합니다.
</Callout>
