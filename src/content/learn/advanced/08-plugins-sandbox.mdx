## 플러그인 & 샌드박싱

이 레슨은 준비 중입니다.

## 플러그인 구조

```
my-plugin/
├── .claude-plugin/
│   └── plugin.json
├── commands/
├── agents/
├── skills/
└── hooks/
```

## 샌드박싱

샌드박싱은 Claude Code의 실행을 안전한 환경으로 격리합니다.

<Callout type="note">
플러그인은 GitHub 저장소로 배포하고 claude plugin install 명령어로 설치할 수 있습니다.
</Callout>
