## Hooks 가이드

이 레슨은 준비 중입니다.

## Hooks 이벤트 타입

| 이벤트 | 설명 |
|--------|------|
| PreToolUse | 도구 실행 전 |
| PostToolUse | 도구 실행 후 |
| SessionStart | 세션 시작 시 |
| SessionEnd | 세션 종료 시 |
| Notification | 알림 발생 시 |

<Callout type="note">
Hooks는 settings.json의 hooks 섹션에서 설정합니다. 훅 스크립트는 shell 스크립트 또는 Python 스크립트로 작성할 수 있습니다.
</Callout>
