## MCP 통합

이 레슨은 준비 중입니다.

## MCP 설정 예시

```json
{
  "mcpServers": {
    "my-server": {
      "command": "npx",
      "args": ["-y", "@my-org/mcp-server"]
    }
  }
}
```

<Callout type="tip">
MCP 서버는 .mcp.json 파일 또는 Claude Code 설정에서 구성할 수 있습니다. 프로젝트별로 다른 MCP 서버를 사용할 수 있습니다.
</Callout>
