## Claude Code CLI 완벽 가이드

이 레슨은 준비 중입니다. 현재 Claude Code 공식 문서의 CLI 레퍼런스 섹션을 참고하세요.

## 기본 명령어 구조

```bash
claude [options] [command]
```

## 주요 플래그

| 플래그 | 설명 |
|--------|------|
| `-p, --print` | 단일 명령 실행 후 종료 |
| `--model` | 사용할 모델 지정 |
| `--verbose` | 상세 출력 모드 |
| `--no-color` | 색상 출력 비활성화 |
| `--version` | 버전 출력 |
| `--help` | 도움말 출력 |
