## 출력 스타일과 Fast 모드

이 레슨은 준비 중입니다.

## 출력 스타일 옵션

<CLIExample
  command="claude --output-format json"
  description="JSON 형식으로 결과를 출력합니다."
/>

## Fast 모드

<CLIExample
  command="claude --fast '간단한 질문'"
  description="더 빠른 모델로 응답을 받습니다."
/>

<Callout type="tip">
빠른 응답이 필요하고 간단한 작업이라면 Fast 모드를 사용하세요. 복잡한 코드 분석이나 리팩토링은 기본 모드가 더 좋은 결과를 제공합니다.
</Callout>
