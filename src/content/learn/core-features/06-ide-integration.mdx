## JetBrains IDE 통합

이 레슨은 준비 중입니다.

## JetBrains 플러그인 설치

1. File → Settings → Plugins 열기
2. Marketplace 탭 선택
3. "Claude Code" 검색
4. Install 클릭 후 IDE 재시작

## 데스크톱 앱 통합

Claude 데스크톱 앱은 VS Code, JetBrains, 기타 에디터와 통합될 수 있습니다.

<Callout type="note">
IDE 통합을 통해 에디터에서 코드를 선택하고 직접 Claude Code에게 질문하거나 수정을 요청할 수 있습니다.
</Callout>
