## 터미널 설정 최적화

이 레슨은 준비 중입니다.

## 권장 터미널

| OS | 권장 터미널 |
|----|------------|
| macOS | iTerm2, Terminal.app |
| Windows | Windows Terminal |
| Linux | GNOME Terminal, Alacritty |

## 셸 설정

Claude Code는 bash, zsh, fish 등 대부분의 셸에서 작동합니다.

<Callout type="tip">
zsh와 oh-my-zsh 조합은 Claude Code와 잘 작동하며, 자동완성과 히스토리 기능을 더 효과적으로 사용할 수 있습니다.
</Callout>
