## Claude Code 메모리 시스템

이 레슨은 준비 중입니다.

## 메모리 계층 구조

Claude Code는 세 가지 수준의 메모리를 제공합니다:

1. **글로벌 메모리**: `~/.claude/CLAUDE.md` - 모든 프로젝트에 적용
2. **프로젝트 메모리**: `./CLAUDE.md` - 현재 프로젝트에 적용
3. **로컬 메모리**: `./CLAUDE.local.md` - 개인 설정 (Git에 커밋 안 함)

<Callout type="tip">
CLAUDE.local.md는 개인 설정이나 로컬 개발 환경에 특화된 정보를 저장하는 데 사용하세요. 이 파일은 .gitignore에 추가하여 팀과 공유하지 않는 것이 좋습니다.
</Callout>

## 효과적인 CLAUDE.md 작성

```markdown
# 프로젝트명

## 기술 스택
- 사용하는 프레임워크, 언어, 도구

## 코딩 규칙
- 팀이 따르는 코딩 스타일

## 자주 사용하는 명령어
- 개발, 테스트, 빌드 명령어
```
