## 체크포인팅과 세션 관리

이 레슨은 준비 중입니다.

## 세션 재개 방법

<CLIExample
  command="claude --continue"
  description="마지막 세션을 재개합니다."
/>

<CLIExample
  command="claude --resume [session-id]"
  description="특정 세션 ID로 세션을 재개합니다."
/>

## 세션 목록 확인

<CLIExample
  command="claude --list-sessions"
  description="이전 세션 목록을 확인합니다."
/>

<Callout type="note">
체크포인팅 기능을 사용하면 장시간 작업 중에 세션이 중단되더라도 작업을 이어갈 수 있습니다. 중요한 작업 전에 Git 커밋을 생성해두는 것도 좋은 방법입니다.
</Callout>
