## 권한 모델 이해

이 레슨은 준비 중입니다.

## 권한 설정 예시

```json
{
  "allowedTools": ["Read", "Write", "Edit", "Bash(npm test)"],
  "disallowedTools": ["Bash(rm -rf)"]
}
```

<Callout type="warning">
권한 설정은 보안에 직접적인 영향을 미칩니다. 최소 권한 원칙에 따라 필요한 도구만 허용하세요.
</Callout>
